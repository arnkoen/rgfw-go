# RGFW-Go

WIP Go bindings for [RGFW](https://github.com/ColleagueRiley/RGFW). Partially machine generated.

**Available build tags:**
- `opengl` - OpenGL support
- `vulkan` - Vulkan support
- `webgpu` - WebGPU support
- `directx` - DirectX support (Windows only)
- (none) - No graphics backend (windowing only)
